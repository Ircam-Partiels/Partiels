# Graphical Workflow Examples
A series of examples that describes some graphical approaches.

## Improve the visualization of a group
1. Export the group as an image, read and analyze the content of the image
2. If necessary, modify the grahical/processor parameters of the tracks with the following requirements 
  * The spectrogram tracks must be behind other tracks (position 0)
  * The visible tracks must use distinct colors 
  * The tracks must use the same colors for the foreground and the text
  * The visible texts on the marker tracks must have different positions
  * The line width and the text font of the tracks must be big enough
  * The visualization should be clear and highlight the analyses
3. Go back to the step #1 until the visualization is satisfying
